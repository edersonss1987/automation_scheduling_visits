# automation_scheduling_visits
Este projeto tem como objetivo automatizar o processo de tarefas pendentes em um sistema CRM (Agendamento de visitas técnicas). 
Utilizando PyAutoGUI, Playwright, Pandas e Integração com API do TELEGRAM.

O script manipula a maquina local, se conecta na VPN, acessa o sistema CRM, BAIXA as tarefas em aberto e as encaminha via Telegram ao responsável.

Motivação:

    1º Extração do endereço, no campo descrição, melhor previsibilidade logística. 
    2º Automarizar processo realizado diariamente.
    3º Facilitar o acesso dos agendamentos
    4º Evitar o esquecimento de impressões de tarefas ou O.S.


Esse projeto pessoal inclui:

    - Integrção com API de TELEBOT (Telegram)
    - Rapagem de dados em sistema Customer Relationship Management ou Gestão de Relacionamento com o Cliente (CRM) .
    - Tratamento de dados usando com PANDAS.
    - Engenharia de atributos
    - Manutenção temporal
